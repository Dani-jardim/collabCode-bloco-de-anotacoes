# Dicas do terminal 
Listando alguns comandos:

- `cd` (para navegar entre pastas)
Como eu entro dentro de uma pasta?
```
d nomeDaPasta
```
Como sair de uma pasta?
```
cd ..
```
- mkdir (comando para criar pastar)

Como crio uma pasta?

```
mkdir nomeDaPasta
```
